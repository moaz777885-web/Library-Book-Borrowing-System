# 📚 Library Book Borrowing System

A C++ console application that simulates a school library book borrowing system,
built using Object-Oriented Programming (OOP) principles.

## 🚀 Features
- Add new books to the library
- Display all available and borrowed books
- Borrow and return books
- Search for a book by ID
- Detect overdue books (borrowed more than 14 days)
- Calculate fines automatically ($0.50 per overdue day)

## 🛠️ Tech Stack
- Language: C++
- Paradigm: Object-Oriented Programming (OOP)
- IDE: Dev-C++

## 📁 Project Structure
| File | Description |
|------|-------------|
| LibraryBook.h | Class declaration and method prototypes |
| LibraryBook.cpp | Class implementation (constructors, destructor, methods) |
| Functions.h | External function declarations |
| Functions.cpp | External function definitions |
| main.cpp | Main function and menu system |

## ⚙️ How to Run
1. Open Dev-C++ and create a new Console Project
2. Add all 5 files to the project
3. Press F9 to Compile & Run

## 📌 OOP Concepts Used
- Encapsulation (private attributes + getters/setters)
- Constructors (default + parameterized)
- Destructor
- Modular programming (separated header and source files)
